## 开发设置

Please make sure your development environment meets the following constraints:

* NodeJS version >= `v16`, you could install a specific version via [nvm](https://github.com/nvm-sh/nvm)

```bash
# 安装依赖包
pnpm install

# 代码检查
pnpm lint

# 启动服务
pnpm dev
```
